¿Qué es mecanicapp?

Mecanicapp es una aplicación web y móvil de gestión vehicular para uso personal, que permite llevar cuenta de los siguientes gastos:

  • por mantenimientos.
  • por lavados.
  • por recargas de combustibles.

Caracterísitcas

Mecanicapp está desarrollada principalmente con la ayuda del framework de desarrollo web Django el cual está basado en el lenguaje de programación Python, además de muchas librerías.
